Workflow

Wat bedoel ik met workflow

Hiermee bedoel ik: Wat ik allemaal heb gedaan om deze site zo te krijgen, hoe ik alle artikellen verander. De tools die ik hiervoor heb gebruikt vind je hier ook terug.

Tools

Voor we beginnen met de codes en andere dingen vind ik het toch zeer belangrijk om te laten weten welke tools ik heb gebruikt. Deze site is gemaakt met de statische website generator HEXO. Deze is vrij gemakkelijk maar is zeker interessant om eens te bekijken. Om al mijn documenten overzichtelijk en ordelijk te houden gebruik ik Brackets.

De installatie

De eerste stap van dit proces is node.js downloaden. De download is te vinden op de website. Open vervolgens de commandprompt (Windows-toets + R —> cmd).

Typ daarna deze code

Hiermee ga je hexo installeren.

Dit is om een project aan te maken. Let wel op dat je op de juiste plaats in je verkenner zit voor je dit doet.

Zo kom je in je werkmap.

Hiermee zet je je server op. Het adres is automatisch ‘Localhost:4000’. Typ dit in je browser en je zal een live preview krijgen van je website.
